1. Simple Coordinates (Lat/Long)

Most common format - used by GPS:
{
  "latitude": 25.0330,
  "longitude": 121.5654,
  "accuracy": 10
}
These coordinates point to Taipei, Taiwan

2. Address Information

Human-readable location:
{
  "street": "123 Zhongxiao East Road",
  "city": "Taipei",
  "state": "Taipei City",
  "country": "Taiwan",
  "postalCode": "10050"
}

3. GeoJSON Format

Standard format for geographic data:
{
  "type": "Feature",
  "geometry": {
    "type": "Point",
    "coordinates": [121.5654, 25.0330]
  },
  "properties": {
    "name": "Taipei 101",
    "category": "landmark"
  }
}
Note: GeoJSON uses [longitude, latitude] order!

4. Browser Geolocation API

What your browser returns:
{
  "coords": {
    "latitude": 25.0330,
    "longitude": 121.5654,
    "altitude": null,
    "accuracy": 20,
    "altitudeAccuracy": null,
    "heading": null,
    "speed": null
  },
  "timestamp": 1737628800000
}

5. Google Maps URL

Location as a shareable link:
https://www.google.com/maps?q=25.0330,121.5654
Click coordinates in Google Maps to get this format

6. IP Geolocation Data

Approximate location from IP address:
{
  "ip": "123.456.789.012",
  "city": "Taipei",
  "region": "Taipei City",
  "country": "TW",
  "loc": "25.0330,121.5654",
  "timezone": "Asia/Taipei"
}
Less accurate than GPS, typically city-level

7. GPS Track Points

Multiple locations with timestamps (tracking movement): 
[
  {
    "lat": 25.0330,
    "lon": 121.5654,
    "timestamp": "2026-01-23T10:00:00Z",
    "speed": 0

  {
    "lat": 25.0335,
    "lon": 121.5660,
    "timestamp": "2026-01-23T10:05:00Z",
    "speed": 5.2
  }
